Tema 2, PILAS

Definición

Una pila es un TDA que sigue el principio de LIFO (Last In, First Out), es decir, el último elemento que se agrega es el primero que se elimina. Las pilas se utilizan para almacenar y manipular elementos de manera ordenada.

Representación en memoria

    Las pilas se pueden representar en memoria de manera estática o dinámica.

  • Arrays (Estatico): Se utiliza cuando el tamaño de la pila es fijo.
  • Listas enlazadas (Dinamica): Se utiliza cuando el tamaño de la pila es variable.

Operaciones básicas

Las operaciones básicas que se pueden realizar en una pila son:

  • Push: agregar un elemento a la pila.
  • Pop: eliminar el elemento superior de la pila.
  • Peek: obtener el elemento superior de la pila sin eliminarlo.
  • IsEmpty: verificar si la pila está vacía.




Aplicaciones

Las pilas tienen una variedad de aplicaciones en la programación en Java, como:

  • Evaluación de expresiones: las pilas se pueden utilizar para evaluar expresiones matemáticas, como la evaluación de una expresión postfix.
  • Análisis de sintaxis: las pilas se pueden utilizar para analizar la sintaxis de un lenguaje de programación, como la verificación de la sintaxis de un programa.
  • Simulación de sistemas: las pilas se pueden utilizar para simular sistemas que involucran la gestión de elementos en una pila, como la simulación de un sistema de colas.

Ejercicio resuelto

Crear un programa que contenga una pila y use los métodos básicos de operación

		
import java.util.Collections;
import java.util.Stack;

public class Pila {

    /**
     * @param args the command line arguments
     */
    public static void main(String[] args) {
        Stack  pila1=new Stack();
        System.out.println(pila1);
        pila1.push(2);
        pila1.push(333);
        pila1.push(100);
        pila1.push(200);
        pila1.push(150);
        pila1.push(3333322);
        System.out.println(pila1 + " ");
        System.out.println("Tamaño de la pila "+pila1.size());
        System.out.println("La busqueda  del elemento es "+pila1.search(333));
        System.out.println("el elemento que se encuentra en la sima es "+pila1.peek());
        System.out.println("Elemento eliminado "+pila1.pop());
        System.out.println(pila1);
        System.out.println("Obtener un elemento "+pila1.elementAt(3));
        System.out.println("el primer elemento es "+pila1.firstElement());
        System.out.println("El ultimo elemento es "+pila1.lastElement());    
        Collections.sort(pila1);
        System.out.println(pila1);
        
    }
}
El resultado de dicho código será: [2, 100, 150, 200, 333]
