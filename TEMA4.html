Tema 4, LISTAS

Definición

Las listas son colecciones dinámicas de elementos que permiten insertar, eliminar y acceder a elementos en cualquier posición. Las listas son otra estructura de datos fundamental en la programación en Java, que nos permite almacenar y manipular elementos de manera ordenada. En este apartado, exploraremos las operaciones básicas, tipos de listas y aplicaciones de las listas en Java.

Operaciones básicas

Las operaciones básicas que se pueden realizar en una lista son:

  • Inserción: agregar un nuevo elemento en una posición específica de la lista.
  • Eliminación: eliminar un elemento de la lista.
  • Consulta: obtener un elemento de la lista sin eliminarlo.
  • Vaciado: eliminar todos los elementos de la lista.

Tipos de listas

Existen varios tipos de listas, cada una con sus propias características y ventajas:

  • Lista simplemente enlazada: es la lista más básica, donde cada elemento apunta al siguiente elemento.
  • Lista doblemente enlazada: es una lista donde cada elemento apunta al siguiente y al anterior elemento.
  • Lista circular: es una lista que se comporta como un anillo, donde el último elemento apunta al primer elemento.

Aplicaciones

Las listas tienen una amplia variedad de aplicaciones en la programación en Java, como:

  • Gestión de datos: las listas se pueden utilizar para almacenar y manipular datos en un orden específico.
  • Procesamiento de texto: las listas se pueden utilizar para procesar texto en un orden específico, como la lectura de archivos o la transmisión de datos.
  • Simulación de sistemas: las listas se pueden utilizar para simular sistemas que involucran la gestión de elementos en un orden específico.

Ejercicio resuelto

Crear un programa que contenga una cola con los colores del arcoíris y aplique las operaciones básicas de las colas

		
import java.util.ArrayList;
import java.util.Collections;

public class ListasEjercicio {
    public static void main(String[] args) {
        ArrayList numeros = new ArrayList<>();
        numeros.add(45);
        numeros.add(12);
        numeros.add(78);
        numeros.add(23);
        numeros.add(56);
        numeros.add(89);
        numeros.add(34);
        numeros.add(67);
        numeros.add(11);
        numeros.add(90);

        System.out.println("Lista antes de ordenar: " + numeros);

        Collections.sort(numeros);

        System.out.println("Lista después de ordenar: " + numeros);

        int primerNumero = numeros.get(0);
        int ultimoNumero = numeros.get(numeros.size() - 1);

        System.out.println("El primer número es: " + primerNumero);
        System.out.println("El último número es: " + ultimoNumero);
    }
}

El resultado de dicho código será:Lista antes de ordenar: [45, 12, 78, 23, 56, 89, 34, 67, 11, 90]
Lista después de ordenar: [11, 12, 23, 34, 45, 56, 67, 78, 89, 90]
El primer número es: 11
El último número es: 90
