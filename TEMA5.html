Tema 5: árboles

Definición

Los árboles son estructuras de datos jerárquicas donde cada nodo tiene un valor y referencias a otros nodos hijos. Se utilizan para representar relaciones jerárquicas y organizadas, como sistemas de archivos. Los árboles son una estructura de datos fundamental en la programación en Java, que nos permite almacenar y manipular elementos de manera jerárquica.

Clásificación de los árboles

  • Árboles generales: Cada nodo puede tener cualquier número de hijos.
  • Árboles binarios: Cada nodo tiene a lo sumo dos hijos (izquierdo y derecho).
  • Árboles binarios de búsqueda: Los valores del subárbol izquierdo son menores que el nodo raíz, y los del derecho son mayores.
  • Árboles balanceados: Mantienen una diferencia mínima entre las alturas de los subárboles, como AVL y árboles rojo-negro.
  • Árbol n-ario: es un árbol donde cada nodo puede tener cualquier número de hijos.

Operaciónes Básicas

Las operaciones básicas que se pueden realizar sobre un árbol binario son:

  • Inserción: agregar un nuevo nodo al árbol.
  • Memoria dinámica.
  • Eliminación: eliminar un nodo del árbol.
  • Consulta: obtener un nodo del árbol sin eliminarlo.
  • Vaciado: eliminar todos los nodos del árbol.
  • Búsqueda: Localizar un nodo específico.

Manera de implementación

En Java, estas operaciones se pueden implementar utilizando métodos como insert(), remove(), get() y clear().

  • En orden (in-order): Visita izquierda, raíz, derecha.
  • Preorden (pre-order): Visita raíz, izquierda, derecha.
  • Postorden (post-order): Visita izquierda, derecha, raíz
  • Por niveles (level-order): Utiliza una cola para recorrer nivel por nivel.

Sintáxis

  • Para insertar: insert();
  • Para remover: remove();
  • Para obtenet: get();
  • Para limpiar: clear();



Aplicaciones

Los árboles tienen una amplia variedad de aplicaciones en la programación en Java, como:

  • Gestión de datos: los árboles se pueden utilizar para almacenar y manipular datos en una estructura jerárquica.
  • Procesamiento de texto: los árboles se pueden utilizar para procesar texto en una estructura jerárquica, como la lectura de archivos o la transmisión de datos.
  • Simulación de sistemas: los árboles se pueden utilizar para simular sistemas que involucran la gestión de elementos en una estructura jerárquica.
  • Sistemas de archivos: Representar carpetas y archivos.
  • Bases de datos: Índices en B-trees y B+trees.
  • Compresión de datos: Árboles de Huffman.
  • IA y videojuegos: Árboles de decisión y árboles minimax.

Ejercicio resuelto

Crear un programa que contenga todos los tipos de ordenes de los árboles

Clase Nodo

		
public class Nodo {
    int dato; // Valor del nodo
    Nodo izquierdo, derecho; // Hijos izquierdo y derecho

    // Constructor
    public Nodo(int dato) {
        this.dato = dato;
        izquierdo = derecho = null;
    }
}

Clase Árbol

		
public class Arbolgral {
    Nodo raiz;
    // Constructor
    public Arbolgral() {
        raiz = null;
    }
    // Recorrido en preorden
    void preorden(Nodo nodo) {
        if (nodo != null) {
            System.out.print(nodo.dato + " "); // Visitar el nodo
            preorden(nodo.izquierdo); // Recorrer el hijo izquierdo
            preorden(nodo.derecho); // Recorrer el hijo derecho
        }
    }
    // Recorrido en inorden
    void inorden(Nodo nodo) {
        if (nodo != null) {
            inorden(nodo.izquierdo); // Recorrer el hijo izquierdo
            System.out.print(nodo.dato + " "); // Visitar el nodo
            inorden(nodo.derecho); // Recorrer el hijo derecho
        }
    }
    // Recorrido en postorden
    void postorden(Nodo nodo) {
        if (nodo != null) {
            postorden(nodo.izquierdo); // Recorrer el hijo izquierdo
            postorden(nodo.derecho); // Recorrer el hijo derecho
            System.out.print(nodo.dato + " "); // Visitar el nodo
        }
    }
}

Clase Main

		
public class Main {
    public static void main(String[] args) {
        Arbolgral arbol = new Arbolgral();
        // Crear el árbol manualmente
        arbol.raiz = new Nodo(1);
        arbol.raiz.izquierdo = new Nodo(2);
        arbol.raiz.derecho = new Nodo(3);
        arbol.raiz.izquierdo.izquierdo = new Nodo(4);
        arbol.raiz.izquierdo.derecho = new Nodo(5);
        arbol.raiz.derecho.izquierdo = new Nodo(6);
        arbol.raiz.derecho.derecho = new Nodo(7);
        System.out.println("Recorrido en Preorden:");
        arbol.preorden(arbol.raiz);
        System.out.println("\nRecorrido en Inorden:");
        arbol.inorden(arbol.raiz);
        System.out.println("\nRecorrido en Postorden:");
        arbol.postorden(arbol.raiz);
    }
}

El resultado de dicho código será:

Recorrido en Preorden:
1 2 4 5 3 6 7
Recorrido en Inorden:
4 2 5 1 6 3 7
Recorrido en Postorden:
4 5 2 6 7 3 1
