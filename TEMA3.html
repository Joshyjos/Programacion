Tema 3, COLAS

Definición

Las colas son una estructura de datos fundamental en la programación en Java, que nos permite almacenar y manipular elementos de manera ordenada.En este apartado, exploraremos la representación en memoria, operaciones básicas, tipos de colas y aplicaciones de las colas en Java. Las colas son estructuras de datos lineales que siguen el principio FIFO (First In, First Out). Esto significa que el primer elemento en entrar será el primero en salir.

Representación en memoria

En Java, una cola se puede representar en memoria utilizando una variedad de estructuras de datos, como arrays, listas enlazadas o vectores.

Las colas pueden representarse mediante:

  • Arrays: Para colas simples y circulares. El tamaño debe definirse previamente, y se necesita lógica para manejar los índices de entrada y salida.
  • Listas enlazadas: Una implementación dinámica donde cada nodo contiene un elemento de la cola y una referencia al siguiente nodo.

Operaciones básicas

Las operaciones básicas que se pueden realizar en una cola son:

  • Inserción ("add()"): agregar un nuevo elemento al final de la cola.
  • Eliminación ("remove()"): eliminar el elemento que se encuentra al principio de la cola.
  • Consulta ("peek()"): obtener el elemento que se encuentra al principio de la cola sin eliminarlo.
  • Vaciado ("clear()."): eliminar todos los elementos de la cola.

Tipos de colas

Existen varios tipos de colas, cada una con sus propias características y ventajas:

  • Cola simple: es la cola más básica, donde los elementos se agregan al final y se eliminan del principio.
  • Cola circular: es una cola que se comporta como un anillo, donde el último elemento apunta al primer elemento.
  • Cola bicola: es una cola que permite la inserción y eliminación de elementos en ambos extremos.

Aplicaciones

Las colas tienen una amplia variedad de aplicaciones en la programación en Java, como:

  • Sistemas de impresión: Los trabajos de impresión se procesan en orden de llegada.
  • Planificación de procesos en sistemas operativos.
  • Gestión de colas en redes y sistemas distribuidos.
  • Gestión de tareas: las colas se pueden utilizar para gestionar tareas que deben ser ejecutadas en un orden específico.
  • Procesamiento de datos: las colas se pueden utilizar para procesar datos en un orden específico, como la lectura de archivos o la transmisión de datos.
  • Simulación de sistemas: las colas se pueden utilizar para simular sistemas que involucran la gestión de elementos en un orden específico.

Ejercicio resuelto

Crear un programa que contenga una cola con los colores del arcoíris y aplique las operaciones básicas de las colas

		
import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

/**
 *
 * @author huawei
 */
public class A {

    /**
     * @param args the command line arguments
     */
    public static void main(String[] args) {
        List coloresArcoiris = new ArrayList<>();
        coloresArcoiris.add("Rojo");
        coloresArcoiris.add("Naranja");
        coloresArcoiris.add("Amarillo");
        coloresArcoiris.add("Verde");
        coloresArcoiris.add("Azul");
        coloresArcoiris.add("Índigo");
        coloresArcoiris.add("Violeta");

        System.out.println("Colores del arcoíris: " + coloresArcoiris);

        int posicionEliminada = 3;
        String colorEliminado = coloresArcoiris.remove(posicionEliminada);
        System.out.println("El color eliminado es: " + colorEliminado);

        System.out.println("Colores después de eliminar uno: " + coloresArcoiris);

        coloresArcoiris.add(posicionEliminada, colorEliminado);
        System.out.println("Colores después de agregar el color eliminado en la misma posición: " + coloresArcoiris);

        Collections.sort(coloresArcoiris);
        System.out.println("Colores ordenados alfabéticamente: " + coloresArcoiris);

        System.out.println("¿La lista contiene 'Verde'? " + coloresArcoiris.contains("Verde"));

        System.out.println("¿La lista está vacía? " + coloresArcoiris.isEmpty());

        System.out.println("Colores del arcoíris:");
        for (String color : coloresArcoiris) {
            System.out.println(color);
        }
    }
}
El resultado de dicho código será: Amarillo Azul Naranja Rojo Verde Violeta índigo
